# Memory-Colors
A classic card memory game online but with colors instead of pictures :)

## How to play
Click on the question marks (the cards) to reveal the color, then click another one to reveal a second color. If the two colors match, then they will stay shown. If they don't match they will both just go back to the question mark state. 

After all cards have been shown and are matched, the game will automatically reset with new colors and a new order!
